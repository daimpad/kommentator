Kommentar-Werkzeug

Textstellen markieren & kommentieren

Einbindbar in jede Seite — als statische Dateien, ohne Server, ohne Build, ohne externe Abhängigkeiten.

Markiere im Text unten eine Stelle mit der Maus (oder per Touch), um sie zu kommentieren. Klick auf eine Markierung oder eine Notiz verbindet beide. Über die Aktionsleiste kannst du eigene Kommentare herunterladen und die exportierten Dateien anderer wieder einlesen.

So bindest du es ein

<!-- 1. Stylesheet in den <head> -->
<link rel="stylesheet" href="kommentare.css">

<!-- 2. Zu kommentierenden Bereich auszeichnen -->
<div data-kommentierbar>
  … dein Fließtext …
</div>

<!-- 3. Skript vor </body> laden und starten -->
<script src="kommentare.js"></script>
<script>
  Kommentare.init({
    container: '[data-kommentierbar]',
    autor: 'Vorname Nachname'
    // optional: margin, toolbar (Mount-Selektoren), readOnly,
    //           texte (eigene UI-Texte / i18n),
    //           onCreate(anno), onUpdate(anno), onDelete(id), onChange(annos)
  });
</script>

Beispieltext

Beispieltext zum Ersetzen

Dies ist Platzhaltertext. Ersetze den Inhalt dieses Bereichs (<div data-kommentierbar>) durch das Dokument, das kommentiert werden soll. Die Markierungs- und Kommentarfunktion arbeitet auf jedem Fließtext, auch über mehrere Absätze und verschachtelte Elemente hinweg.

Jede Markierung speichert den exakten Wortlaut, etwas Kontext davor und danach sowie die Zeichenposition im Text. So lassen sich Kommentare beim Wiedereinlesen genau an derselben Stelle verankern — vorausgesetzt, der Ausgangstext bleibt zwischen den Runden unverändert.

Für geteilte Sichtbarkeit ohne Server sammelst du die exportierten Dateien der Beteiligten ein und liest sie über „Kommentare laden“ gemeinsam ein. Alle Notizen erscheinen dann nebeneinander in der Randspalte, mit Namen der jeweiligen Autor:innen.
